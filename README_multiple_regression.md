
# Multiple Linear Regression – Multi-Channel Marketing Analysis

## Project Overview
This project uses Multiple Linear Regression to evaluate the effect of TV, Radio, and Social Media marketing spend on Sales.

## Files Included
- multiple_regression_analysis.ipynb
- marketing_sales_data.csv
- README_multiple_regression.md

## Technologies Used
- Python
- pandas
- seaborn
- matplotlib
- statsmodels

## Setup

Install required libraries:

```bash
pip install pandas numpy matplotlib seaborn statsmodels scipy notebook
```

## Analysis Workflow
1. Load and clean data
2. Perform EDA
3. Detect multicollinearity using VIF
4. Build regression model
5. Validate assumptions
6. Interpret business results

## Business Insight
Channels with statistically significant positive coefficients should receive higher marketing budget allocation.
